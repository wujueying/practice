1.2 手机屏幕现状
- 移动端设备屏幕尺寸非常多，碎片化严重
- 安卓设备有多种分辨率：480x800、480x854、540x940、720x1280
- 近年来iPhone的碎片化也加剧了，其设备的主要分辨率有：640x960，640x1136，750x1334
- 作为开发者无需关注这些分辨率，因为我们常用的尺寸单位是px。
